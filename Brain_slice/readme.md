## 3D Brain Slice Phantom

This repository contains STL files for a modular 3D-printable brain slice phantom. The phantom is designed for imaging experiments and can be easily assembled and customized.

### Files

- **`Brain_phantom_3DSlice.stl`**  
  Main body of the brain slice phantom.

- **`Brain_phantom_3DSlice_Cover.stl`**  
  Cover piece to seal the phantom.

- **`Brain_phantom_3DSlice_Plug.stl`**  
  Plugs for closing the filling openings.

### Description

The phantom consists of a brain-shaped slice with GM, WM and CSF segments. The cover ensures a secure seal, while the plugs allow flexible opening and closing of filling ports.
