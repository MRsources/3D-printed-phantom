# 3D Printed Phantom

This repository provides STL files for a modular, 3D-printable phantom designed for MRI experiments and prototyping.

---

### Segmentation Data

These files describe the anatomical structure used to design the phantom:

- **`Brain_slice`**  
  https://github.com/MRsources/3D-printed-phantom/tree/main/Brain_slice  
  Segmented brain slice including Gray Matter (GM), White Matter (WM), and Cerebrospinal Fluid (CSF).
